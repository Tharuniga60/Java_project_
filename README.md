\# Java Project - CodingShare



A simple Java Swing + MySQL based project that handles user login and registration.



\## Features

\- User authentication (Login \& Register)

\- MySQL database integration

\- Clean Swing UI



\## Database

Database: `codingshare`  

Tables: `users`, `resources`



\## Technologies

\- Java (JDK 17 or above)

\- MySQL

\- JDBC

\- VS Code
